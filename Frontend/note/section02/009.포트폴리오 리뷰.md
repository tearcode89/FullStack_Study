로직의 흐름 이해하기

1. 입력폼에 입력값 입력
2. submit 버튼 클릭
3. 바인딩된 이벤트 핸들러 함수 실행
4. 이벤트 발생
5. 검증 로직 실행
6. set****Error 의 변화
7. onChangeWriter onChangePassword onChangeTitle, onChangeContents 에 있는 if문 로직은 submit 버튼이 눌리고 에러가 발생한 상황에서 재입력시 작동하게 만든 로직
